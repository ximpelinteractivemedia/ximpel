Syntax - Overlays

Overlays in XIMPEL are clickable elements that can be shown during a media item. Overlays are often used to allow the user to choose which subject he wants to jump to. An example is shown below:

playlist.xml

						<?xml version="1.0" encoding="utf-8"?>
						<ximpel>
						<playlist>

						    <subject id="subject1">
						        <media>
						            <video>
						                <source file="video_filename1" extensions="mp4" types="video/mp4" />
										<overlay startTime="3" leadsTo="subject2" shape="rectangle" x="0px" y="0px" width="300px" height="200px" text="Go to subject2!" />
						            </video>
						        </media>
						    </subject>
						 
						    <subject id="subject2">
						        <media>
						            <video>
						                <source file="video_filename2" extensions="mp4" types="video/mp4" />
										<overlay startTime="3" leadsTo="subject1" shape="oval" x="0px" y="0px" width="300px" height="200px" text="Go to subject1!" />
						            </video>
						        </media>
						    </subject>

						</playlist>
						</ximpel>
In this example there are two subjects, both with one video. Both of these videos defined an overlay to be shown after the first three seconds of that video has played. When you click on on the overlay of the video of subject1 you will jump to subject2. If you click on the overlay from the video of subject2 you will jump to subject1 again.

Multiple overlays

Any number of overlays can be displayed at any point during a media item (video/audio/picture/etc). You can specify when the overlay should appear and when the overlay should disappear using the startTime and duration attributes respectively. Omitting the duration attribute will keep the overlay visible until another media item or subject is started. 
						<picture src="test_files/pictures/demo.png">
						    <overlay startTime="0" duration="1" x="0px" y="0px" width="150px" height="150px" backgroundColor="red" />
						    <overlay startTime="1" duration="1" x="150px" y="0px" width="150px" height="150px" backgroundColor="red" />
						    <overlay startTime="2" duration="1" x="300px" y="0px" width="150px" height="150px" backgroundColor="red" />
						 
						    <overlay startTime="0" duration="1" x="300px" y="150px" width="150px" height="150px" backgroundColor="red" />
						    <overlay startTime="1" duration="1" x="150px" y="150px" width="150px" height="150px" backgroundColor="red" />
						    <overlay startTime="2" duration="1" x="0px" y="150px" width="150px" height="150px" backgroundColor="red" />
						</picture>


Conditional leadsTo

As with subjects you can specify conditional leadsTo values for overlays so that when the overlay is clicked, depending on a condition it will jump to one subject or another. An example is given below, but for more explanation see: subjects and variables.

playlist.xml

						<subject id="subject1" leadsTo="subject1">
							<media>
								<picture src="myimage.png" leadsTo="subject1">
									<overlay leadsTo="subject1" startTime="0" x="0px" y="0px" width="150px" height="150px" >
										<leadsTo subject="subject2" condition="{{score1}} > 8" />
										<leadsTo subject="subject3" condition="{{score1}} > 6" />
									</overlay>
								</picture>
							</media>
						</subject>


Overview of supported attributes and elements

There are many other attributes available for the overlay. A list is shown below.
Attribute name Possible values Description
x For example: 100px The x coordinate the overlay will appear at
y For example: 100px The y coordinate the overlay will appear at
width For example: 200px The width of the overlay
height For example: 200px The height of the overlay
shape rectangle, oval, circle, square What the shape of the overlay should be
sides For example: 200px The length of the sides of a square overlay (only used when overlay has shape "square".
diameter For example: 200px The diameter of the overlay (used when overlay has shape "circle")
leadsTo For example: introSubject The name of the subject to jump to when this subject finishes.
startTime For example: 200px The time in seconds from the start of the video when the overlay should appear.
duration For example: 5 The number of seconds the overlay should be shown. Not specifying this or using value 0 means the overlay is shown until the media item is terminated.
text For example: "Click me!" The text to show on the overlay
alpha Between 0 and 1, for example: 0.6 The transparency of the overlay (0 is fully transparent, 1 is fully opaque)
hoverAlpha Between 0 and 1, for example: 0.6 The transparency of the overlay while hovering over it (0 is fully transparent, 1 is fully opaque)
backgroundColor For example: "red" or #000000 The background color of the overlay
hoverBackgroundColor For example: "green" or #000000 The background color of the overlay while hovering over it
textColor For example: "green" or #000000 The text color of the overlay
hoverTextColor For example: "green" or #000000 The text color of the overlay while hovering over it
fontFamily For example: "arial" The font to use on the overlay.
hoverFontFamily For example: "arial" The font to use on the overlay when hovering over it.
fontSize For example: "16pt" or "2em" or 50px The text size of the overlay
hoverFontSize For example: "16pt" or "2em" or 50px The text size of the overlay while hovering it.
image For example: images/picture.png The background image to use for the overlay
hoverImage For example: images/picture.png The background image to use for the overlay while hovering it
